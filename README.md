# Neuro-Karaoke-Song-Adder
Specialized tool for adding new songs to the Neuro Karaoke Archive V3

I have included a color_config.json file featuring the default color settings.
To use custom colors, simply place a copy of color_config.json in the same directory as the executable (.exe) and edit the values to your preference.
